== NAME

xhc-hb04-accels - Obsolete script for jogging wheel

== SYNOPSIS

*xhc-hb04-accels*

== DESCRIPTION

*xhc-hb04-accels* Obsolete script, xhc-hb04.tcl now controls reduced
wheel jogging accels.
