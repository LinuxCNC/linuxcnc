= iocontrol(1)

== NAME

iocontrol - interacts with HAL or G-code in non-realtime

== SYNOPSIS

*[EMCIO]*

== DESCRIPTION

I/O control handles I/O tasks like coolant, toolchange and E-stop. The
signals are turned on and off in non-realtime with G-code or in the case
of E-stop in HAL. +
