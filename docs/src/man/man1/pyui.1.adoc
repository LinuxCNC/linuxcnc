= pyui(1)

== NAME

pyui - utility for panelui

== SYNOPSIS

*loadusr pyui*

== DESCRIPTION

*pyui* Validates panelui.ini files

This will read, try to correct, then save the panelui.ini file. It will
print errors to the terminal if found.
