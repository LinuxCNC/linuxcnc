= debuglevel(1)

== NAME

debuglevel - sets the debug level for the non-realtime part of LinuxCNC

== SYNOPSIS

*debuglevel* *-ini* _INIFILE_

== DESCRIPTION

*debuglevel* displays a checkbox to select the current debug level
of some parts of LinuxCNC.

== SEE ALSO

halcmd(1) - debug subcommand linuxcnc(1)*

Much more information about LinuxCNC and HAL is available in the
LinuxCNC and HAL User Manuals, found at /usr/share/doc/LinuxCNC/.
