About QtWoodPecker

If you select a file from the help_files folder that ends with .html, it will be shown here.

Any HTML files chosen from a folder other than help_files is assumed to be a setup sheet and will be shown in the UTILS page under the SETUPS tab.

This is useful for displaying setup sheets created by CAM programs that generate HTML setup files.

As linuxcnc is primarily a cnc controller and not a web browser, link navigation is enabled only for files in the help_files folder. All other html files have link navigation disabled. It is therefore advisable to place setup sheets in a different location because they could potentially contain links to javascripts which could cause fatal errors in WoodPecker.
