## Implementation Plan: Go Launcher Phase 1a

**Current progress (March 2026):**
- Milestones M1, M2, M3, M4, M8, and M9 are implemented and validated.
- TPMOD/HOMEMOD preload now mirrors legacy `scripts/linuxcnc.in`; motmod linking resolved (PR #137).
- The Go launcher successfully starts realtime, preloads motion modules, and executes HAL files.
- INI parser, lockfile, flag handling, and config tests are fully functional.
- Legacy bash launcher remains unaffected (can run in parallel).

---

### Milestones & Tasks (Updated Status)

#### M1: Project Bootstrap
- [x] Create Go module and project structure (`src/launcher/`)
- [x] Compile-time config flags via `-ldflags`
- [x] Command-line argument parsing (support legacy flags: -d, -v, -l, -k, -r, -t, -m, -H)
- [x] Lockfile management (`/tmp/linuxcnc.lock`)

#### M2: Custom INI Parser
- [x] INI parsing: support `#INCLUDE`, `[SECTION]KEY` lookup, repeated keys
- [x] Support config fallback via pickconfig.tcl (stub: call as subprocess)
- [x] Variable substitution in HAL files
- [x] `IniFile.Set()` for in-memory INI mutation (PR #136)

#### M3: HAL File Executor ✅ (PR #134)
- [x] HAL file loading with HALLIB_PATH resolution
- [x] INI variable substitution in HAL file lines
- [x] Execute HAL files via `halcmd -i <ini> -f <file>` subprocess
- [x] HALFILE/HALCMD interleaving in INI-file order
- [x] POSTGUI_HALFILE support
- [x] LIB: prefix handling

#### M4: Realtime Manager ✅ (PR #135)
- [x] Absorb `realtime.in` script logic (uspace-only)
- [x] Check `/dev/zero` is writable
- [x] Start validation (uspace no-op)
- [x] Shutdown via `rtapi_app exit` + poll loop + IPC cleanup (`ipcrm`)
- [x] Force-kill fallback (`pkill -KILL rtapi_app`)
- [x] Zombie process filtering in `IsRunning()`

#### Motmod Preload Fix ✅ (PR #136 → PR #137)
- [x] Resolve TPMOD: CLI `-t` → `[TRAJ]TPMOD` → default `tpmod`
- [x] Resolve HOMEMOD: CLI `-m` → `[EMCMOT]HOMEMOD` → default `homemod`
- [x] Pre-load via `halcmd loadrt tpmod` + `halcmd loadrt homemod` (separate subprocesses)
- [x] `[EMCMOT]EMCMOT` INI value is NOT mutated (stays as `motmod`)
- [x] Regression tests for resolution priority chain

#### M5: Process Manager ⏳ *(next — currently blocking)*
