Select the Object to copy from Qtvcp's builtin selection.

Copy Base Name

This is the name used for folders and files. It can be the same as Qtvcp's builtin name.

You can change it if you wish to be able to switch between the builtin and a customized version.

Whatever base name you use, this would be the name you use in the INI to load a screen or the name used to load the panel in a HAL file
